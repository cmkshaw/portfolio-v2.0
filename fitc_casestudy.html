Case Study: FITC



This was a very fun project I whipped up for FITC’s women’s initiative contest this year.

I decided that a short, one page explanation of why I wanted to go to FITC would be a great way to demonstrate my creative abilities.



This project was simple, but successful in that it communicated my intentions while providing a visually pleasing experience.

Development

I used a parallax effect so that it would be fun and visually appealing when the user is scrolling through the page. I also used jQuery so that the user has to interact (click) sections of the website in order to display more information.
